Domain Relaxing

Domain relaxing is a method of enabling web-based scripting between components that the web browser would normally prevent due to cross-domain scripting security concerns. In some cases, however, this scripting enables desired functionality. Domain relaxing is designed to allow cross-domain scripting between components served from similar root domains (i.e. server1.co.com and server2.co.com). Help Menu Integration for SAP NetWeaver Enterprise Portal uses the same domain relaxing technique that SAP uses.

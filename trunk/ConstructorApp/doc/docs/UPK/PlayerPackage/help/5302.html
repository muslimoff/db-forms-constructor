Запуск проигрывателя

Чтобы получить доступ к содержанию, нужно запустить проигрыватель. Ваш администратор предоставит вам интернет-адрес для запуска проигрывателя. Когда вы пройдете по этому адресу, появится ссылка Player.

Если у вас есть вопросы о пользовании проигрывателем, нажмите в окне проигрывателя для отображения интернет-справки.

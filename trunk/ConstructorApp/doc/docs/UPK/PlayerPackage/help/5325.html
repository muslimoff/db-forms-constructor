Know It? Типы действий и событий режима

В зависимости от текущего действия вы можете получить инструкции для выполнения этапа в Режиме Know It?. Например, если вам требуется ввести текст в поле или выбрать определенное значение из списка, текст может появиться в пузыре Know It?, который обеспечивает руководство на этом этапе.

В режиме изучения также поддерживается автоматическое завершение шагов в зависимости от типа действия. Если в процессе разработки контента для иллюстрации выделения текста или перемещения объекта по экрану использовалось перетаскивание мышью, такой шаг в режиме изучения будет выполнен автоматически и отмечен как выполненный правильно. В пузыре «Изучение» выводится следующий текст: «Это действие выполняется путем перетаскивания. Если нажать ОК, данный шаг будет выполнен автоматически. После его завершения вы можете перейти к следующему шагу. За этот шаг баллы начислены не будут». Если нажать кнопку «ОК», задание будет завершено автоматически в режиме изучения. После выполнения действия с перетаскиванием можно продолжить работу над заданием.

Для большей части этапов пузырь Know It? просто отображает ссылку Показать этап, без инструкций по выполнению этапа.

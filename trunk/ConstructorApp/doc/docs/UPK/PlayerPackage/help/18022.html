Воспроизведение в режиме See It!

Режим See It! позволяет ознакомиться с просмотром анимированного показа заданий, выполняемых в искусственной среде. При выполнении заданий предоставляется информация, объясняющая произведенные действия.

Воспроизведение можно в любой момент приостановить, щелкнув ссылку «Пауза» в пузыре. Когда просмотр потребуется продолжить, щелкните ссылку «Возобновить» для продолжения воспроизведения. Также можно в любой момент остановить воспроизведение, нажав кнопку «Закрыть» в пузыре, или вручную перейти к следующему шагу, нажав клавишу ENTER . Для выхода достаточно нажать ESC.

Для управления воспроизведением темы используется ссылка на действия, которая появляется в пузыре.

Ссылка

Описание

Возобновить

Закрыть мен действий и возобновить воспроизведение темы.

Печать

Отображает документ, предоставленный по теме вашим провайдером содержания. Ссылка появляется только если документ предоставлен для режима Print It!

Параметры

Отображение текущих параметров проигрывателя. Если ссылка не появляется, это значит, что ваш администратор установил параметры заранее и исключил возможность их изменения.

Справка

Отображение системы справки проигрывателя.

Закрыть тему

Закрывает воспроизведение текущей темы и возвращает к точке выбора тем.

Чтобы воспроизвести тему в режиме See It!:

  1. Запустите проигрыватель.
  2. Разверните обзор так, чтобы отобразить темы.
  3. Щелкните тему. Можно также дважды щелкнуть тему мышью, чтобы воспроизвести ее в режиме default playback mode.
  4. Нажмите .

    Вы можете также начать тему с определенного пункта Jump-In Point.

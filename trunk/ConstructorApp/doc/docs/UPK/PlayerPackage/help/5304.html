Настройка параметров проигрывателя

Окно параметров можно использовать для настройки работы проигрывателя.

Примечание. Если значок параметров не показан в окне проигрывателя, это означает, что администратор уже установил все параметры и заблокировал их, чтобы исключить возможность изменения.

Настройки параметров по умолчанию можно восстановить, нажав «По умолчанию» и затем нажав «ОК» для подтверждения.

Параметр

Описание

Цвет рамки

Установите цвет рамки (или выделения), которая появляется в режимах See It!, Try It! и Do It!. Цвет рамки по умолчанию - красный.

Воспроизвести звук
  • Все доступные звуки (по умолчанию) - воспроизводятся все доступные звуковые файлы.
  • Клавиатура и щелчки мышью - при воспроизведении в режиме просмотра воспроизводятся только нажатия клавиш и щелчки мышью. Эти звуки представлены файлами FLV, включаемыми в папку Audio опубликованного контента; они воспроизводятся автоматически при нажатии клавиши или щелчке мышью в режиме просмотра.
  • Без звука - во время воспроизведения звук отключается.
    Примечание. Для воспроизведения звука в проигрывателе необходимо установить Flash 9.0 или более позднюю версию.

Показать вводный текст

Этот параметр определяет, где появится рамка с введением, когда тема будет воспроизводиться в проигрывателе. Выберите этот параметр для отображения рамки введения как основной и отмените выбор для того, чтобы ее сжать.

Разрешает перемотку в режиме Try It!

Выберите этот параметр, чтобы разрешить переход к следующему этапу в режиме Try It! нажатием клавиши Ввод вместо выполнения действия в пузыре.

Сочетание клавиш для работы в режиме Do It!

Настройте комбинацию клавиш для перехода к следующему шагу в режиме выполнения, если не хотите использовать комбинацию по умолчанию.

Режим воспроизведения по умолчанию

Этот параметр управляет режимом, который работает по умолчанию при двойном щелчке по теме в обзоре. Режимом по умолчанию является режим Try It!

Для настройки параметров проигрывателя:

  1. Запустите проигрыватель.
  2. Нажмите .
  3. Настройте параметры проигрывателя.
  4. Нажмите кнопку «ОК».

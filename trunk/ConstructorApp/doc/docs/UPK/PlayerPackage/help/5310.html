Фильтрация содержания по роли

Панель обзора включает ссылку My Roles, если ваше содержание включает назначение ролей. Окно ввода Мои роли и ссылка Мои роли появляются только если вы просматриваете содержание, в котором для автора содержания назначаются роли.

Функция Мои роли позволяет вам персонализировать содержание, чтобы отображать только те темы, которые соответствуют роли. Например, вы можете выбрать «Платежная ведомость», чтобы отображать только те темы, которые относятся к функции платежной ведомости.

Щелкните ссылку Мои роли и выберите роли, которые будут использоваться для фильтрации содержания, из заранее введенного списка в диалоговом окне Мои роли. Щелкните окошко Роли, чтобы применить к содержанию фильтрации на основе ролей. На панели обзора появятся только те документы, для которых назначены выбранные роли. Вы можете легко переключаться между персонализированным просмотром и полным просмотром содержания. Вы можете заново отобразить полный обзор, закрыв окошко Мои роли.

Примечание. При использовании фильтрации на основе ролей вам может понадобиться развернуть дерево структуры в обзоре, чтобы увидеть результаты фильтрации.

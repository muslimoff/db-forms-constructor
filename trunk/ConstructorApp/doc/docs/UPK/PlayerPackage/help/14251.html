Соответствующий параметр

Параметр «Соответствующий контент» задает вывод контента, относящегося к экрану, который в данный момент отображается в выбранном приложении. Этот параметр можно использовать для поиска контента, включающего информацию по текущему экрану. Такая возможность позволяет уменьшить объем контента, из которого требуется выбирать материалы для просмотра. Этот объем можно еще больше сократить (в списке тем соответствующего контента), используя текстовое поле Поиск.

Примечание. Доступность параметра «Соответствующий контент» зависит от установленного источника контента. Если параметр доступен, он отображается над полем поиска на панели обзора.

Для использования Соответствующего параметра:

  1. Запустите проигрыватель.
  2. Щелкните параметр «Соответствующий контент», чтобы вывести список материалов, относящихся к открытому в данный момент экрану выбранного приложения.
  3. Воспроизведите тему в любом режиме, или щелкните какой-либо модуль или раздел, показанный на экране.

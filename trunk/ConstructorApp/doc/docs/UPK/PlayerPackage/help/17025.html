Использование режима Print It!

Режим Print It! отображает документ, который ваш провайдер содержания приложил к теме. Обычно это вспомогательный документ или другой документ с соответствующей информацией по теме. Документ откроется либо в Microsoft Word, либо в Adobe Acrobat, в зависимости от формата. Вы можете ознакомиться с документом интерактивно или напечатать его.

Примечание. Можно также использовать режим Print It! из меню действий в пузыре в тех режимах воспроизведения, которые это поддерживают.

Чтобы использовать режим Print It!:

  1. Запустите проигрыватель.
  2. Разверните обзор так, чтобы отобразить темы.
  3. Щелкните тему. Можно также дважды щелкнуть тему мышью, чтобы воспроизвести ее в режиме default playback mode.

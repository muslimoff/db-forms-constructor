Воспроизведение темы

Можно воспроизвести тему в любом из следующих режимов (если их предоставляет ваш провайдер содержания):

  • Просмотр
  • Тест
  • Выполнение
  • Изучение
  • Печать

Примечание. Если в проигрывателе публикуется веб-страница со звуком, связанная с панелью информации темы, вверху панели информации проигрывателя появляются значки воспроизведения и паузы, позволяющие контролировать воспроизведение звука.

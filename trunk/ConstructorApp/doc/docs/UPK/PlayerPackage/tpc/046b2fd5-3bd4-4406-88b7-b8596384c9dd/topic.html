Создание табличной формы (грид)

Данный пример демонстрирует создание простой формы-таблицы в конструкторе форм.

Для выбора сервера:

И выберите необходимый вам сервер из появившегося списка.

Введите имя пользователя.

Введите пароль.

Для создания дочернего узла.

Если включен режим отладки, появится сообщение об успешном выполнении операции с частью выполнившейся PL/SQL процедуры. Для просмотра полного текста вызова процедуры и переданных в нее и полученных параметров параметров необходимо нажать иконку детализации в верхней правой части окна сообщения.

После просмотра данных отладочного сообщения нажмите кнопку OK.

Обновите данные дерева.

Будет осуществлен переход в режим редактирования.
